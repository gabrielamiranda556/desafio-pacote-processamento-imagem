# processamento_imagem

## Contexto do Projeto

Este repositório faz parte de um desafio educacional da DIO com foco no aprendizado da criação e empacotamento de pacotes Python aplicados ao processamento de imagens.

O objetivo principal do projeto é compreender na prática a estrutura de empacotamento, incluindo a organização de diretórios, definição de dependências (requirements.txt), configuração de arquivos como setup.py e a documentação do projeto.

O desenvolvimento foi baseado em um repositório de referência disponibilizado durante o curso, seguindo boas práticas utilizadas em projetos Open Source. O foco do desafio não está na complexidade do processamento de imagens, mas sim na construção, organização e publicação de um pacote Python funcional e reutilizável.

----------------------

Description. 

The package processamento_imagem is used to:
	Processamento:
		- Comparacao de histogramas
		- Similaridade de estruturas
		- Redimensionamento de imagem
	Utilidades:
		- Carregar imagem
		- Salvar imagem
		- Exibir imagem
		- Exibir resultados
		- Exibir histograma

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install dio-desafio-pacote-processamento-imagem

```bash
pip install dio-desafio-pacote-processamento-imagem
```
